# Elite Web Development

Use `/elite-web` to activate high-end web development mode. Focus on premium aesthetics, clean architecture, and modern Next.js patterns.

## Tech Stack
- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS v4 (CSS-first)
- **UI Components**: shadcn/ui (customized for premium look)
- **Motion**: Framer Motion
- **Icons**: Lucide React

## Premium Design Rules
- **Color**: Use deep, sophisticated palettes (e.g., `#09090b` for backgrounds, vibrant accent gradients). No generic colors.
- **Typography**: Inter or Geist. Use high contrast for headings, subtle opacity for secondary text.
- **Surfaces**: Glassmorphism (blur: 20px, border: 1px white/10%).
- **Interaction**: Hover scales (1.02x), subtle border glows, and spring-based animations.

## Next.js Best Practices
- **Server Components**: Default to RSC. Use 'use client' only for interactivity.
- **Data Fetching**: Use Server Actions for mutations. Parallel routes for complex UIs.
- **Performance**: Optimize images, use font optimization, and minimize client-side JS.

## Commands
- `/elite-web-init`: Scaffolds a premium Next.js project.
- `/elite-web-component`: Generates a high-end UI component with motion.
