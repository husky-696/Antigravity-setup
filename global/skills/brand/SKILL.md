# Brand Identity & Copy

Use `/brand` to transform a coding project into a professional brand instantly. Focus on identity, voice, and visual storytelling.

## Branding Protocol
1. **Logo Generation**: Use `generate_image` to create high-end, minimalist logos and app icons.
2. **Color Palette**: Design a cohesive color story (Primary, Accent, Surface, Background) that fits the vibe.
3. **Typography**: Select a font pairing (Heading + Body) from Google Fonts or system defaults.
4. **Copywriting**: Generate taglines, "About" sections, and landing page copy that sounds professional and elite.

## Best Practices
- **Minimalism**: Focus on clean, high-impact visuals.
- **Consistency**: Ensure the palette and type are used across the entire app.
- **Voice**: Maintain a senior, confident, and direct tone in all marketing copy.

## Commands
- `/brand-init`: Generates a full brand kit (logo, colors, copy) for a new idea.
- `/brand-logo`: Focuses on iterating and refining the app icon/logo.
- `/brand-copy`: Generates landing page or store description copy.
