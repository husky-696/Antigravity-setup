# Ether Vault (Personal)

Hardened patterns for building secure, biometric-enabled local vaults in Expo/React Native.

## Core Patterns
- **Biometric Locking**: Implementation of `expo-local-authentication` with fallback logic.
- **Secure Storage**: Using `expo-secure-store` for sensitive keys.
- **Local-First Sync**: Patterns for syncing secure data with Supabase without exposing raw values.

## How to Invoke
- Use `/ether-vault` when working on secure mobile storage.
