# Elite Design & Aesthetics

Use `/elite-design` to elevate the visual quality of any project. Focus on luxury, clarity, and motion.

## Core Design Tokens
- **Typography**: Strictly use Inter, Geist, or Outfit. Focus on `text-balance` and varied weights.
- **Colors**: Curated dark modes (`#09090b`), vibrant gradients (Indigo to Violet), and high-contrast accents.
- **Surfaces**: Glassmorphism (blur: 20px), thin borders (white/10%), and subtle inner shadows.
- **Motion**: Use Framer Motion or Reanimated with spring physics. No linear animations.

## Aesthetic Patterns
- **Card Luxury**: Rounded corners (16px+), subtle border glows on hover, and smooth entry transitions.
- **Micro-interactions**: Scale effects (1.02x), haptic feedback, and button states that feel tactile.
- **Layout**: Generous whitespace, centered content where appropriate, and consistent 4px grid.

## Commands
- `/elite-design-review`: Audits current UI and suggests premium improvements.
- `/elite-design-tokens`: Generates a Tailwind or CSS variable theme based on a vibe.
