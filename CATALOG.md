## ⚡ Common Workflows

### 🚀 New Feature Build
- Use `/nextjs-app-router-patterns` for the architecture.
- Apply `/stitch-design-taste` for the UI design.
- Use `/shadcn` for component implementation.
- Test with `/android_ui_verification` (if mobile).

### 🐛 Systematic Debugging
- Start with `/debug-systematic` to isolate the cause.
- Use `/bug-hunter` to trace root-cause to specific lines.
- Verify fix with `/code-review`.

### 🤖 Agent Orchestration
- Use `/ai-agents-architect` for the system design.
- Implement with `/langgraph` or `/crewai`.
- Optimize performance with `/agent-orchestration-multi-agent-optimize`.
