## ⚡ Elite Workflows

### 🎨 Build a Premium Feature
1. Activate `/elite-web` or `/elite-mobile`.
2. Use the **Premium Design Rules** from the skill to draft the UI.
3. Implement with `/elite-backend` for data persistence.

### 🐛 Resolve a Critical Bug
1. Activate `/debug`.
2. Follow the **Isolate → Trace → Analyze** protocol.
3. Ship the fix with `/elite-ship`.

### 🚀 Initialize New Project
1. Use `/setup-init` to audit the environment.
2. Activate necessary `/elite-*` skills based on the stack.
3. Start building with `/elite-web-init` or `/elite-mobile-init`.
